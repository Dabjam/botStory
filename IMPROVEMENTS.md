### 2. **3D Робот с Three.js**

#### Компонент: `Robot3D.tsx`

**Возможности:**
- ✨ Полноценная 3D модель робота с использованием React Three Fiber
- 🎭 Реалистичная анимация: покачивание, моргание глаз, движение антенны
- 💡 Динамическое освещение с цветными spotlight'ами
- 🌟 Particle system (искры вокруг робота)
- 🔮 Энергетические кольца с прозрачностью
- 🎨 Металлические и эмиссивные материалы
- 🎮 Интерактивные OrbitControls (вращение мышью, зум колесиком)
- 🪞 Отражающая поверхность (ground plane)
- 🌃 Environment mapping для реалистичных отражений

**Детали модели:**
- Голова с движущейся анимацией
- Моргающие неоновые глаза (розовые)
- Антенна с колебанием и светящимся шариком
- Ядро энергии в груди (пульсирует)
- Сочлененные руки и ноги
- Энергетические кольца-ауры

**Материалы:**
- Тело: Зеленый металл (#00ff9f) с эмиссией
- Голова: Голубой металл (#00d9ff) с высокой эмиссией
- Глаза: Яркий пурпурный (#ff00ff) toneMapped=false
- Антенна: Желтый (#ffff00) с максимальной яркостью

**Использование:**
```tsx
import Robot3D from '../components/Robot3D'

<Robot3D />
```

**Где применен:**
- 🏠 Landing page (главная страница) - заменил простую 2D модель

## 🐛 Известные Ограничения

1. **Шрифт Minecraft** - требует ручной установки (copyright)
2. **Three.js** - требует WebGL поддержку (старые браузеры не поддерживаются)
3. **Производительность** - 3D робот может быть медленным на слабых устройствах

### Решения:
- Для старых устройств можно отключить 3D робота (fallback на 2D)
- Minecraft шрифт имеет fallback на Orbitron
- Canvas анимации адаптируются под fps
