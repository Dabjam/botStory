# 🚀 Руководство по Обновлению до v2.1

## ⚡ Быстрый Старт

### 1. Обновите зависимости
```bash
cd frontend
npm install
```

### 2. Добавьте шрифт Minecraft (опционально, но рекомендуется)

#### Скачайте шрифт:
- 🔗 **Основной источник**: https://www.dafont.com/minecraft.font
- 🔗 **Альтернатива**: https://fontmeme.com/fonts/minecraft-font/

#### Установите:
```bash
# Поместите скачанный minecraft.ttf в:
frontend/public/fonts/minecraft.ttf
```

### 3. Запустите проект
```bash
# Из корня проекта
docker-compose up --build

# ИЛИ только фронтенд для разработки
cd frontend
npm run dev
```

### 4. Откройте в браузере
```
http://localhost:5173
```

## 🐛 Возможные Проблемы и Решения

### Проблема: 3D робот не отображается
**Решение:**
1. Проверьте консоль браузера на ошибки WebGL
2. Убедитесь что у вас современный браузер (Chrome 90+, Firefox 88+, Safari 14+)
3. Проверьте что видеокарта поддерживает WebGL 2.0

### Проблема: Minecraft шрифт не применяется
**Решение:**
1. Убедитесь что файл `minecraft.ttf` находится в `frontend/public/fonts/`
2. Очистите кеш браузера (Ctrl+Shift+R)
3. Перезапустите dev-сервер

### Проблема: Медленная производительность
**Решение:**
1. Закройте другие вкладки браузера
2. Отключите расширения браузера
3. На слабых устройствах можно уменьшить zoom карты

### Проблема: npm install выдает ошибки
**Решение:**
1. Удалите `node_modules` и `package-lock.json`
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```
2. Проверьте версию Node.js (требуется 18+)
   ```bash
   node --version
   ```

- **Браузер**: Chrome 100+, Firefox 100+, Safari 15+
